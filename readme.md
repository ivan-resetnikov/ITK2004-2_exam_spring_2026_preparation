# Get started

## Prerequisites

- Python 3 installed
- That's it. No other dependencies.

## Running

It's simple, just run `__main__.py` file found in the repository's root. How you do it depends on how you installed python, but form most people it's:

### Linux:
- Run `$ cd "path/to/project" && python3 main.py`

### Windows:
- Double click on the `__main__.py` file
- OR run `> cd "path\to\project" && python main.py`
